API Endpoints

Complete reference for all available endpoints and parameters

Search Songs
GET
/api/songs/search
  • query - Search term (required)
  • language - Filter by language
  • year - Filter by release year
  • sort - Sort by popularity/latest/duration
  • limit - Number of results (max 50)
Search Albums
GET
/api/albums/search
  • query - Search term (required)
  • language - Filter by language
  • year - Filter by release year
  • sort - Sort by popularity/latest
  • limit - Number of results
Search Artists
GET
/api/artists/search
  • query - Artist name (required)
  • limit - Number of results
Search Playlists
GET
/api/playlists/search
  • query - Search term (required)
  • language - Filter by language
  • sort - Sort by popularity/latest
  • limit - Number of results
Song Details
GET
/api/songs/details
  • id - Song ID (required)
Artist Top Songs
GET
/api/artists/top
  • id - Artist ID (required)
